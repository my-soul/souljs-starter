souljs-starter

## Description

[souljs](https://github.com/my-soul/souljs) framework TypeScript starter repository.

[soul-orm](https://github.com/my-soul/soul-orm) mysql toolkit

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start
```
